About

Introducing the Vanila Music Player which is built with HTML, CSS and JavaScript having Fuild like experience with your favourite songs on the list. We like things to be simple because simplicity leads towards elegance. A music player for every one.

Features

We added the features which are required the most and skip the useless features added to new music players now a days in order to avoid the complexity and start trends towards simplicity.

  • Play
  • Pause
  • Skip Next
  • Skip Back
  • Select Random
We Know what exactly you need because Vanilla Music Player is from Developer for Developers so that you can code with your favourite beat.
